# 🍔 FoodExpress - MERN Food Delivery App

A full-stack food delivery web application built using the MERN stack (MongoDB, Express, React, Node.js).

## 🚀 Features
- User authentication (JWT)
- Browse restaurants & food items
- Add to cart & place orders
- Admin dashboard
- Coupon system (coming soon)

## 🛠 Tech Stack
- Frontend: React.js, Vite
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: CSS / Tailwind

## 📁 Project Structure
/client → React frontend  
/server → Node.js backend
